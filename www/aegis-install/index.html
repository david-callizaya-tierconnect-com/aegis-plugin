AEGIS PLUGIN

Some Things You Should Know About it

To be Able to Record and Set Definitions to the visited pages, you need to install the Aegis-Plugin. Please, use FIREFOX , click on install when ever it appears install. Once you install, restart the FireFox to be able to setup the initial configuration. On the top navigation bar of the FIREFOX explorer, look for the AEGIS Icon. Click the icon to set the ip's corresponding to the BackEnd Server and Front-End Server and Agis Plugin Server. Click save and Restart the Explorer.
