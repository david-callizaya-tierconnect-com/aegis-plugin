AEGIS PLUGIN

Some Things You Should Know About it

To be Able to Record and Set Definitions to the visited pages, you need to install the Aegis-Plugin, to do so follow these steps:

  • Open FIREFOX, click on the install button, do this every time “Install” appears once the installation is complete, restart FIREFOX and proceed with the initial configuration setup.
  • On the top navigation bar of FIREFOX, look for the AEGIS24 Icon. Click on the icon to set the IP's corresponding to the Back-End, Front-End and the Aegis Plugin Servers (Use the values provided on the image).
  • Click save and Restart FIREFOX.
